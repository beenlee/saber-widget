0.2.0 / 2014-06-16
==================

* **移除** 模块 `lib`, API 移植到 `saber-lang/type`
* `Widget` **增加** 新API `plugin`、`enablePlugin`、`disablePlugin`

* **增加** 插件基类 `Plugin`, 及新插件 `Masker`、`Zoom`
* 更新文档及DEMO
